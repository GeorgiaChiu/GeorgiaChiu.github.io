穿戴裝置 x 舞蹈

Wearing Installation x dancing

Video

Description

形式：

舞蹈裝置演出

作品論述：

『少了什麼會活不下去？』

人們心裡必定有個分量特別大的什麼來維持每天的生活，他像於石窟深處歷經時光滄桑所沈澱出的渾厚氣場，倘若崩塌，生活就像粉塵碎屑，飄無定所且生無可戀。 『若是信仰不存在，那這個社會會不會崩壞？』

這個演出，藉由電磁鐵通電和斷電，把人吸在牆面上，製造人可以漂浮的錯覺。

With a waering installation which is strong enough to stick people on a wall by an electromagnet, the dancer seems to float in the air, interpretating like a godness.

向光Follow the lights

穿戴裝置 x 接觸即興

Wearing Installation x Contact improvisation

Video

Description

形式：

穿戴式聲音裝置實驗

作品論述：

與人接觸，不論是物理性抑或非物理性，當肢體有了碰撞，我好奇，這會有怎麼樣的變化。

經由我右大腿上配置的一台自製控制器（聲音控制的選鈕以及錄製按鈕裝於其上），配上蔓延在自己手工編織的螢光外衣上的處壓感測，若特殊接觸點被他人所觸動，即有聲音的反饋（Puredata即時演算）。 藉由這些，我想把自己變成一個音樂的觸動裝置，讓他人可以和我一起探討接觸中的奧妙

When you touch the particular part of my waering device, there's sth happen: you made some sound!

This installation is designe for me to be like an musical instrument. When people touch me, the sound will change a bit.

My desire to make this project is to create an new interaction of people while doing contact improvisation.

I always wonder the mysterious things between sound and physical contact, the non-verbal language and reality.
